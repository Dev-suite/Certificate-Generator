API

In the very near future, Certificate Magic will have an API (Application Programming Interface) for web developers, allowing the download of certificates from your own website, by using the Certificate Magic certificate generator.

Using just a couple of lines of simple HTML, you will be able to integrate Certificate Magic’s certificate generator into your own site, allowing you to produce professional looking certificates without any need for complex coding.

You will also be able to add your own logo to the certificates to make them uniquely yours.

The API facility will be available very soon for a nominal subscription fee. If you would like to be notified when the API facility is available, please leave your email address below.
